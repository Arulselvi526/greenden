# greenden Website
A modern and responsive plant store website built using HTML and Tailwind CSS. The project features a clean and user-friendly interface with a Home Page, Product Page, and Contact Page. Designed with a mobile-friendly layout, it showcases plant collections, product listings, and an easy way for users to get in touch.

This project helped me strengthen my skills in HTML, Tailwind CSS, responsive web design, and building multi-page websites with a professional and visually appealing user experience.

# Features
Responsive design for mobile, tablet, and desktop devices
Home page with attractive hero section and featured content
Product page showcasing plant collections
Contact page with user-friendly contact form
Built using HTML and Tailwind CSS
Clean and modern UI design

# Technologies Used
HTML5
Tailwind CSS
Responsive Web Design
Flexbox & Grid Layouts

# Purpose
This project was created to practice frontend web development fundamentals and improve website design and styling skills.
